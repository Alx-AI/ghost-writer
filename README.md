# - Ghost Writer -
A project exploring building a GPT-Neo powered writer's assistant, trained on books of 6 different categories:
#### Romance
#### Dystopian
#### Historical Fiction
#### Fantasy
#### Horror
#### Science Fiction

using aitextgen and pytorch.
https://ghost-writer.ai-camp.org/

Deployed with Flask
Built with Nicepage..
Completed in 3 weeks
